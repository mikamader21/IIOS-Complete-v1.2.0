# IIOS Weekly Release Watch

**Cadence:** Monday morning, America/Guayaquil  
**Notification policy:** only meaningful changes

## Sources monitored

- Hermes Agent releases/security/docs.
- Anthropic models, Claude Code and Cowork changes.
- Obsidian release/security/sync/headless changes.
- Graphify releases/security/breaking changes.

## Meaningful change

New stable release, security advisory, deprecation, breaking configuration, pricing/rate-limit change, new governance capability, data-retention change or incompatibility affecting IIOS.

## Report format

Component, old/new, date, severity, IIOS impact, recommended action, urgency and official source.

## Upgrade rule

Watch does not auto-upgrade. Every material upgrade follows staging, backup, tests, ADR where architectural and Owner approval for production.
