# IIOS Foundation v1.2.0

IIOS es un sistema privado de inteligencia institucional gobernado por el Owner. Hermes Agent, Claude Code, Claude Cowork, proveedores de modelos, Obsidian y Graphify son componentes subordinados y reemplazables.

## Estado de esta versión

La v1.2.0 incorpora la auditoría independiente de Claude Cowork y corrige:

- frontera de seguridad específica para Cowork;
- definición técnica verificable del Invariant Kernel;
- separación correcta entre Governance, Orchestrator y Audit;
- comportamiento fail-closed;
- registro externo de modelos y precios volátiles;
- permisos iniciales de Claude Code;
- workflow de verificación para GitHub;
- propiedad del Control Center;
- etiquetado del Vault como espejo no autoritativo.

Las versiones, precios y capacidades externas no son autoridad documental. Se mantienen en `config/model-registry.json` y deben verificarse antes de uso real.

## Fases

La versión actual autoriza ratificación documental, controles de repositorio, especificación del Governance Core y pruebas locales. No autoriza trading, pagos, retiros, compras, despliegues autónomos ni promoción automática de skills.
